# VIM MODE 


## Mode List 

1. `Normal Mode `
2. `Visual Mode `
3. `Visual block Mode `
4. `Visual line Mode `
5. `Command-Line Mode `
6. `Insert Mode `
7. `Replace Mode `
8. `Terminal Mode`



## How to enter different mode and some common key 

1. `Normal Mode`
    - **ENTER:**  
        - `ESC` 或 `ctrl + [`

2. `Command-line Mode`
    - **ENTER:** 
        -  `ESC` 或 `ctrl + [`
    - **traverse command:** 
        - `ctrl + p` 或 `ctrl + n`
    - **jump the begining or ending of the cmd:** ``
        - `ctrl + b` 和 `ctrl + e`
    - **open command-line window**
        - `ctrl + f` 或 `q:`
