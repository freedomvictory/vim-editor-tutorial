# EASY MOTION 


## Instruction 

It's a vim extension and simplify the process of text browsing.


## Install it 

The github repo address is *https://github.com/easymotion/vim-easymotion*


## How to use it 

`[leader_key][leader_key]` + 

- `w`
- `f`
- `F`
- `t`
- `w`
- `b`
- `e`
- `ge`
- `k`
- `j`
- `n`

I think you should just try the above command .
